# OCR Analyzer (TrOCR + FastAPI + React)

A full-stack OCR application that extracts text from images using deep learning (TrOCR) and displays detected regions visually.

---

## Features

- OCR for images
- Transformer-based OCR (TrOCR)
- Multi-line text detection
- Bounding box visualization
- Clean dark UI
- FastAPI backend + React frontend

---

## Tech Stack

### Backend
- FastAPI
- OpenCV
- PyTorch
- HuggingFace Transformers (TrOCR)

### Frontend
- React (Vite)
- Axios
